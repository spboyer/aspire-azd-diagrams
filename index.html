Diagram 3: Data Flow — azd aspire deploy

Step-by-step: user runs azd aspire deploy, extension generates manifest, provisions, deploys.

sequenceDiagram
    participant Dev as 👩‍💻 Developer
    participant AzdExt as 🟢 azd aspire deploy
(extension)
    participant DotNet as 🔧 dotnet CLI
    participant AzdCore as 🔵 azd core
(gRPC services)
    participant Azure as ☁️ Azure

    Dev->>AzdExt: azd aspire deploy

    Note over AzdExt,DotNet: Step 1: Generate Manifest
    AzdExt->>DotNet: dotnet run --publisher manifest
--output-path /tmp/manifest/
    DotNet-->>AzdExt: manifest.json + Bicep files

    Note over AzdExt: Step 2: Parse Manifest
    AzdExt->>AzdExt: Parse manifest.json
Extract services, resources
Evaluate expressions

    Note over AzdExt,AzdCore: Step 3: Get Azure Context
    AzdExt->>AzdCore: gRPC: EnvironmentService
    AzdCore-->>AzdExt: Subscription, RG, location
    AzdExt->>AzdCore: gRPC: AccountService
    AzdCore-->>AzdExt: Azure credentials

    Note over AzdExt,Azure: Step 4: Provision Infrastructure
    AzdExt->>Azure: Deploy Bicep (ACA env OR App Service plan,
PostgreSQL, Redis, Service Bus, etc.)
    Azure-->>AzdExt: Resource outputs + endpoints

    Note over AzdExt,Azure: Step 5: Build + Push Containers
    AzdExt->>AzdCore: gRPC: ContainerService
    AzdCore->>Azure: Build images, push to ACR
    Azure-->>AzdCore: Image digests
    AzdCore-->>AzdExt: Push complete

    Note over AzdExt,Azure: Step 6: Deploy Services
    AzdExt->>Azure: Deploy to Container Apps / App Service
    Azure-->>AzdExt: Deployment complete

    Note over AzdExt,Dev: Step 7: Report Results
    AzdExt-->>Dev: ✅ Deployed to rg-myapp-dev
🌐 https://api.myapp.dev
